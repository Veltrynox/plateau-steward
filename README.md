# Plateau Steward
The character is a maintenance steward assigned to a chain of remote mineral plateaus on an ocean planet where almost nothing breaks the surface.
A study project in Unity attempting to recreate core Subnautica-style systems and mechanics

## Scattering Tool
A Scattering Tool for placing objects on a terrain.
* Hot-Swappable Palette: Instant brush switching using number keys (1-9).
